Create venv (virutal envoriment) and then install the libraries.

Windows 11 or 10
You must have Python version 3.11.9 (This version is tested)
1. python -m venv venv
2. Set-ExecutionPolicy Unrestricted -Scope Process 
3. \venv\Scripts\activate or .\\venv\Scripts\activate
4. pip install -r requirements.txt


If there are some errors left try to install this 


pip install python-dotenv
